# HA Input Bridge

Home Assistant custom integration for controlling a Windows PC through HA Input Bridge.

This integration exposes Home Assistant services for mouse movement, mouse clicks, scrolling, keyboard input, and hotkeys. It is designed to work with the PC Trackpad Card and a Windows bridge agent running on the target PC.

## Status

Experimental.

This project is still being built. The Home Assistant integration is usable for testing, but the Windows installer and polished setup flow are still planned.

## Architecture

Home Assistant
→ HA Input Bridge integration
→ Windows HA Input Bridge agent
→ Windows mouse and keyboard input

Optional dashboard card:

Mobile Home Assistant dashboard
→ PC Trackpad Card
→ HA Input Bridge services
→ Windows PC

## Requirements

- Home Assistant
- HACS
- A Windows PC running the HA Input Bridge agent
- Network access from Home Assistant to the Windows PC
- A shared secret token generated by the Windows bridge

## Installation through HACS

1. Open HACS.
2. Open the three-dot menu.
3. Select Custom repositories.
4. Add this repository URL.
5. Select category Integration.
6. Install HA Input Bridge.
7. Restart Home Assistant.
8. Go to Settings → Devices & services → Add integration.
9. Search for HA Input Bridge.
10. Enter host, port, token, and name.

## Configuration

The setup flow asks for:

| Field | Description |
| --- | --- |
| Name | Display name for this bridge |
| Host | IP address or hostname of the Windows PC |
| Port | Bridge port, default `8765` |
| Token | Secret token used by the Windows bridge |

Example:

Host: 192.168.2.2
Port: 8765
Token: your-generated-token

## Services

After setup, this integration provides these services:

ha_input_bridge.arm
ha_input_bridge.position
ha_input_bridge.move
ha_input_bridge.move_relative
ha_input_bridge.click
ha_input_bridge.scroll
ha_input_bridge.write
ha_input_bridge.press
